# lexiguard-ai
Contract analysis system with chatbot and reporting
##  Description
LexiGuard AI is a contract analysis system that helps identify risk levels in agreements and provides structured insights.

##  Features
- Clause-level risk analysis
- Dashboard with metrics and charts
- Contract upload (PDF/DOCX)
- AI chatbot for contract-related queries
- PDF report generation

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- ReportLab
- OpenAI API

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
